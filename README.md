# Proposta Comercial OXY EcoSystem

Proposta interativa desenvolvida pela **2STEP MÍDIA** para a OXY EcoSystem, focada em estratégias digitais premium para importação de carros e aeronaves de luxo.

## 🚀 Deploy no Vercel

### Método 1: Conectar repositório GitHub
1. Suba os arquivos para um repositório no GitHub
2. Acesse [vercel.com](https://vercel.com) e conecte sua conta GitHub
3. Clique em "New Project" → "Import Git Repository"
4. Selecione o repositório e clique "Deploy"

### Método 2: Vercel CLI
```bash
npm i -g vercel
vercel
```

### Método 3: Arrastar e soltar
1. Acesse [vercel.com](https://vercel.com)
2. Arraste a pasta do projeto diretamente no dashboard
3. Deploy automático será iniciado
